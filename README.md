## Features
- Admin panel
- Question bank
- Student login
- Real-time results

## Requirements
- PHP >= 7.4
- MySQL
- XAMPP or similar local server

## Setup
1. Clone the repo
2. Import `oesm.sql` into your MySQL
3. Run the project via XAMPP
